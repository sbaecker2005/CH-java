Documentação dos Endpoints — Inteligência Artificial

Análise de Lead

POST /api/ai/leads/{id}/analisar
→ retorna recomendação estratégica + score de conversão (200 OK / 404)
POST /api/ai/urgencia
→ classifica texto como CRITICO / ALTO / NORMAL (200 OK)

Modelo Preditivo (Data Science)

GET /api/leads/urgentes
→ leads classificados como urgentes pelo scoring
GET /api/leads/dashboard
→ KPIs consumidos pelo modelo ML offline
PATCH /api/leads/{id}/status
→ atualizar status (alimenta retreino do modelo)
A documentação completa OpenAPI/Swagger está disponível em /swagger-ui.html.
