Detalhes do Pipeline
Stack
  • Python 3.11 + scikit-learn 1.3
  • pandas, numpy, matplotlib, seaborn
  • joblib (serialização)
Pré-processamento
  • StandardScaler para variáveis numéricas
  • OneHotEncoder(handle_unknown='ignore')
  • Train/test split estratificado 75/25
Modelos comparados
  • LogisticRegression(max_iter=2000)
  • DecisionTreeClassifier(max_depth=6) ← vencedor
  • RandomForestClassifier(n=300, depth=10)
  • KNeighborsClassifier(k=11)
Validação
  • Métricas no test set (n=125)
  • 5-fold StratifiedKFold no treino (n=375)
  • F1-Score como métrica principal (target desbalanceado)
